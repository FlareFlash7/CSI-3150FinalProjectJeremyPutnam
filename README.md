project overview 
	The application I have chosen to create is option 3. Option 3 is a trivia application. 
  The trivia application is able to solve the issue of studying and attempting to remember important information. 
  The application can help the user study and learn randomly chosen facts.
  
Step 1: To start the application, click on the gitHub repository link.
Step 2: Choose from the dropdown menu for categories. The categories are general knowledge, science, history and film. 
Step 3: Choose from the dropdown menu for difficulty. The choices are easy, medium and hard.
Step 4: Click the “start game” button to start the trivia.
Step 5: Read and answer the question within the 15 second time limit. Repeat this process until the trivia is over. There will be 50 questions.
Step 6: Review what score you have gotten and compare to previous scores.
